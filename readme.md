# Namaste React – Learning Journey 🚀

This repository documents my journey of learning **React.js** through the **Namaste React course by Akshay Saini**.

The purpose of this repository is to **track my progress, organize my notes, practice concepts, and document mistakes and learnings** while building a strong foundation in React and modern frontend development.

I am still early in the course, and this repository will continue to grow as I progress through the lectures.

---

## 📚 About the Course

**Course:** Namaste React  
**Instructor:** Akshay Saini

The course focuses on understanding **how React works internally** rather than just building projects. It emphasizes strong fundamentals and teaches concepts step-by-step.

Some of the topics covered in the course include:

- React Fundamentals
- JSX
- Components
- Props
- State
- React Hooks
- Rendering & Reconciliation
- Performance Optimization
- Building Real-World Projects
